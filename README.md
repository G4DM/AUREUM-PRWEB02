# AUREUM-PRWEB02

AUREUM-PRWEB02 is a web interface design project focused on responsive layout development. Built primarily with Tailwind CSS and custom CSS using media queries, it emphasizes structure, adaptability, and clean visual design, with minimal JavaScript usage. The project follows a mobile-first approach and is deployed on Netlify for live preview.

## ⚙️ Installation & Setup

1. Clone the repository:

```bash
git clone https://github.com/G4DM/AUREUM-PRWEB02.git
```

2. Install dependencies:

```bash
npm install
```

3. Run development server:

```bash
npm run dev
```

4. Open the local URL provided by Vite in your browser.
