# Declaración sobre la realización del proyecto web

El proyecto web ha sido desarrollado siguiendo las siguientes directrices y referencias:

## 1. Estructura y componentes principales
- El **header/nav** y el **footer** han sido implementados utilizando **TailwindCSS** y fueron diseñados con **asistencia de IA**, aprovechando su sistema de utilidades para asegurar un diseño responsive, moderno y consistente.  
- La implementación de la fuente tipográfica principal se ha realizado siguiendo las indicaciones obtenidas a través de **Claude AI**, asegurando uniformidad en todo el proyecto.  
- Paleta de colores extraída de [coolors.co](https://coolors.co/palette/0f0f0f-1c1c1c-c9a24d-e6d8a8-f5f2ea). Se ha decidido utilizar una paleta de colores con altos contrastes blancos y negros, utilizando como color "diferenciador" toques dorados, para expresar modernismo y lujo.  
- La mayoría de imágenes de este proyecto están extraídas de [Pixabay](https://pixabay.com/es/), y las que no se ha asegurado de que estén libres de derechos de autor.  
- Se ha usado también [FontAwesome](https://fontawesome.com/), en concreto **la versión 6.7.2,** para maximizar compatibilidad.

## 2. Referencias de diseño e inspiración
- La estética y estructura de la web se han basado en varias referencias de alto nivel en el sector gastronómico y de experiencias culinarias:  
  - [Osteria Francescana](https://osteriafrancescana.it/)  
  - [Pujol](https://pujol.com.mx/)  
  - [La Barraca](https://labarraca.nl/)  
  - [The Fat Duck](https://thefatduck.co.uk/)  
- Adicionalmente, se ha tomado como referencia un **propio proyecto previo de e-Commerce** desarrollado con WordPress ([mianet.es](https://mianet.es/)), principalmente para la estructura de servicios.

## 3. Metodología de desarrollo
- La web se ha construido **partiendo de la base del proyecto anterior InterRed**, ajustando, cambiando y añadiendo secciones y estilos según las necesidades del nuevo proyecto.  
- En ciertos casos, cuando los estilos no estaban claros o no se encontraban soluciones previas en Internet, Reddit o Stack Overflow, se consultó a la **IA** para obtener orientación o soluciones puntuales.

## 4. Optimización y organización de archivos
- Inicialmente, el proyecto contaba con **tres archivos CSS independientes** para los distintos archivos `.html`.  
- Posteriormente, se decidió **unificar todo el CSS en un único archivo**, utilizando la asistencia de la IA para reorganizar y optimizar los estilos, manteniendo la coherencia visual y la eficiencia del proyecto.

## 5. Despliegue
- El proyecto se encuentra actualmente desplegado en Netlify y es accesible a través del siguiente enlace: [prweb02-aureum.netlify.app](https://prweb02-aureum.netlify.app)

---

En conjunto, el proyecto combina técnicas modernas de desarrollo frontend, responsive, referencias de diseño profesional y la asistencia de herramientas de IA para lograr un producto visualmente coherente, funcional y escalable.
